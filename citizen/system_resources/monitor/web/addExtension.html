Coming soon...

When I started txAdmin (then FXAdmin) it was just a simple tool to help the co-owner of my server start/stop scripts.

Now, after a month, txAdmin is capable of easily integrating everything from FXServer to Web interface and Discord servers. And I think this is a fertile ground for innovation, so we decided to transform txAdmin into a FXServer Management Framework.

Soon we will provide an unified interface and documentation for extension developers to start helping us take this project to the next level.

Some planned extensions:
  • VRP Explorer: Visualize all your players in one screen. You will be able to see their money, groups, cars and inventory contents.
  • txMaestro: Unify multiple servers under only one control panel.

If you are as excited about the future of this project as we are, join our Discord Server and help us discuss and develop it.
