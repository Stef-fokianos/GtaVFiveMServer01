Σχολή οδήγησης


Καλώς ήρθατε στη σχολή οδηγών.


Όλοι οι πολίτες του Λος Σάντος πρέπει να περάσουν τα τέστ για να μπορούν να οδηγήσουν.
Πάρτε το χρόνο σας, απαντήστε με κοινή λογική, μην απαντήσετε στην τύχη.

Εξετάσεις για τα σήματα
- Ο έλεγχος για τα σήματα κοστίζει 200 ​​$, δεν θα επιστραφεί εάν αποτύχετε.
- Μην φοβάστε, η σχολή οδηγών δέχεται πιστώσεις, αλλά προσέξτε να μην υπερχρεωθείτε
- Εάν χάσετε τη δοκιμή σας για πρώτη φορά, δεν μπορείτε να την επαναλάβετε αμέσως,
θα πρέπει δοκιμάστε ξανά άλλη μέρα.
Εξέταση οδήγησης
- Η δοκιμή οδήγησης κοστίζει 500 $, ενώ για τη δοκιμή κώδικα, αυτή η πληρωμή δεν θα επιστραφεί εάν αποτύχεις.
- Το κύριο πράγμα για να πάρετε την άδειά σας είναι να δώσετε προσοχή σε ό, τι σας περιβάλει και πάνω απ 'όλα αποφύγετε τα ατυχήματα!
